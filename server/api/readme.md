## あばうと
* PHP 5.6
* Laravel 5.1
* Windowsでも構築できるけど面倒

## 最初にやること

### Composerパッケージインストール
`composer install`

### npmパッケージインストール
`npm install`

## 起動の仕方
`php artisan serve`
http://localhost:8000/ にアクセス
